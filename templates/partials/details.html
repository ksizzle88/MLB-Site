I chose to use baseball data since I know that it is a rich, well maintained, freely available, large data set.

The first step in the process was to get the data from the MLB.

The data is freely available at http://gd2.mlb.com/components/game/mlb/ however it needed to be compiled into a more manageable format that the thousands of xml documents that it exists in publicly.

Rather than use an the only api i could find available PitchFX data (PitchRX) I chose to write my own script because I wanted to use Python, and design the database in SQLAlchemy so it would be easier to use in conjunction with flask.

After looking at the data to better understand the structure I designed a database using MySql in conjunction with python and SQLAlchemy.

I wrote a script using beautifulsoup to navigate the pages, and save all of the data to the database.

Unsure of the analysis I eventually wanted to do, I figured I’d play around with the data a little bit. After plotting all of the pitches thrown to David Ortiz, I became interested in seeing where the de facto strike zone is in comparison with the strike zone as defined by the rulebook.

Next I filtered out all pitches that were not swung at, marking the strikes as red and the balls as blue.

The data from the MLB came with strike zone data (the height of the top and the bottom of the strike zone as defined by the rulebook) for each pitch. Next I took the average top and bottom for each pitch on the plot and and drew a rectangle using those points (the left and right edges are defined by home plate which is 17 inches wide). It was clear that more strikes are called down and away than up and in.

It was at this point that I had enough to begin sharing. I abstracted my code so I could apply it to any chosen player, and built this website using flask.

I plan on doing more analysis on this data, in particular, to answer the question “where is the strike zone in reality?”

I plan on doing analyses using K nearest neighbors, SVM, Logistic regression, and perhaps neural network.

I understand that there is going to be a clear bias in this data, specifically that I am only looking at pitches are not swung at, perhaps there are few strikes called up and in on David Ortiz
